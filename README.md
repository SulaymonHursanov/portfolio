# Хурсанов Сулаймон — Portfolio

Personal portfolio website of **Sulaymon Khursanov**, Senior Java Developer with 8+ years of experience building high-load backend systems.

🌐 **Live:** [sulaymonhursanov.github.io/portfolio](https://sulaymonhursanov.github.io/portfolio)

---

## Stack

- **Framework:** Next.js 16 (static export)
- **Styling:** Tailwind CSS v4
- **UI:** shadcn/ui
- **Animations:** Framer Motion, Motion
- **Language:** TypeScript
- **Deploy:** GitHub Pages via GitHub Actions

## Sections

- **Hero** — Bio, highlights, contact links
- **Skills** — Tech stack with glowing card effects
- **Experience** — Work history with animated borders

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

Static output is generated in the `/out` folder.
